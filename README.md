# AnatomyB — תרגול אנטומיה ב׳

Interactive practice-quiz app for the Tel Aviv University "אנטומיה ב'" course (abdomen, pelvis & perineum, lower limb).

- 461 exam-style multiple-choice questions, sourced from course summaries and past-exam recollections, each with an explanation and a citation back to the source document/page.
- Default mode simulates the real exam structure: 60 questions (20 אגן, 20 בטן, 20 גף תחתון), drawn randomly from the pool on every run.
- A custom-practice mode lets you pick topics, question count, and ordering freely.

No build step — static HTML + React (loaded via CDN) + plain JS data files. Just open `index.html`, or deploy as a static site.
